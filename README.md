# MVC-Estudiantes
 
# Ejercicio de clase práctica de Modelo-Vista-Controlador (MVC)

Este repositorio contiene una pequeña aplicación de ejemplo construida como ejercicio para la clase Plataformas de Programación Empresarial 2026-1, creada por Camilo José Ardila Restrepo. El ejercicio ilustra los conceptos básicos del patrón de diseño Modelo-Vista-Controlador (MVC) usando Java.

Esta es una implementación sencilla del patrón MVC, donde:

Modelo representa las clases y datos de negocio,

Vista simple en HTML usando Thymeleaf,

Persistencia usando un ArrayList en la clase EstudiantesDB,

Controlador tipo RESTful que gestiona las interacciones entre la Vista y el Modelo.


# Requisitos:

Antes de compilar o ejecutar el proyecto, asegúrate de tener instalado:

JDK 21 (versión recomendada y usada, cambiar desde configuración si es necesario)

Gradle
